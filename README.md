# 🏐 Escalação de Times - Haikyuu!!

Um projeto web interativo para os fãs de vôlei e de anime montarem o seu próprio time dos sonhos de Haikyuu!! 

Este web app permite escalar jogadores informando o nome e o número da camisa, gerando uma tabela dinâmica com a foto do personagem automaticamente.

## 🚀 Funcionalidades
* **Adição de Jogadores:** Insira o nome (ex: Hinata, Kageyama) e o número da camisa para escalar.
* **Fotos Dinâmicas:** O sistema reconhece o nome do jogador e carrega sua foto correspondente.
* **Remoção de Jogadores:** Botão interativo "X" na tabela para dispensar um jogador específico do time.
* **Design Moderno:** Interface com tema escuro elegante (estilo All Black) e detalhes em laranja, construída com Flexbox.

## 💻 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização e layout responsivo.
* **JavaScript (Vanilla):** Lógica de programação e manipulação do DOM para criar, atualizar e remover elementos dinamicamente na tela.
